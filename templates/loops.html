Loops in Python

Loops in Python allow you to repeatedly execute a block of code. Python supports two types of loops:

  • For Loop: Used to iterate over a sequence (e.g., lists, strings).
  • While Loop: Used to repeatedly execute a block of code as long as a condition is true.

Examples:

For Loop:

        
fruits = ['apple', 'banana', 'cherry']
for fruit in fruits:
    print(fruit)  # Output: apple, banana, cherry

While Loop:

        
count = 0
while count < 5:
    print(count)  # Output: 0, 1, 2, 3, 4
    count += 1
