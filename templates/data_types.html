Data Types in Python

Python supports several data types to represent different kinds of values. Here are some common data types:

  • Integer: Whole numbers (e.g., 1, 2, -3)
  • Float: Numbers with decimal points (e.g., 3.14, -0.5)
  • String: Textual data enclosed in single or double quotes (e.g., 'Hello', "Python")
  • Boolean: True or False values

Examples:

Integer:

        
x = 5
y = -10
print(x)  # Output: 5
print(y)  # Output: -10

Float:

        
pi = 3.14159
temp = -2.5
print(pi)   # Output: 3.14159
print(temp) # Output: -2.5

String:

        
name = 'Alice'
message = "Hello, Python"
print(name)    # Output: Alice
print(message) # Output: Hello, Python

Boolean:

        
is_valid = True
has_error = False
print(is_valid) # Output: True
print(has_error) # Output: False
