Conditionals in Python

Conditional statements in Python allow you to control the flow of your program based on certain conditions. Python supports the following conditional statements:

  • if: Used to execute a block of code if a condition is true.
  • else: Used to execute a block of code if the same condition is false.
  • elif: Short for "else if," used to test multiple conditions.

Examples:

if Statement:

        
age = 18

if age >= 18:
    print("You are an adult")  # Output: You are an adult
else:
    print("You are not an adult")

elif Statement:

        
score = 85

if score >= 90:
    print("A")
elif score >= 80:
    print("B")  # Output: B
elif score >= 70:
    print("C")
else:
    print("F")

Nested if Statements:

        
x = 10

if x > 0:
    if x % 2 == 0:
        print("Positive and even")  # Output: Positive and even
    else:
        print("Positive and odd")
else:
    print("Non-positive")
