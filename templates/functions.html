Functions in Python

Functions in Python allow you to encapsulate a block of code and reuse it throughout your program. Here's how you define and use functions:

Example:

Function Definition:

        
def greet(name):
    """This function greets the person passed in as a parameter"""
    print("Hello, " + name + "!")

# Function Call
greet("Alice")  # Output: Hello, Alice!

Returning Values:

        
def add(a, b):
    """This function adds two numbers and returns the result"""
    return a + b

# Function Call
result = add(5, 3)
print(result)  # Output: 8

Default Parameter Values:

Python allows you to specify default values for function parameters. These defaults are used when the function is called without providing a value for that parameter. For example:

        
def greet_with_default(name='Guest'):
    """This function greets the person passed in as a parameter or 'Guest' if no name is provided"""
    print("Hello, " + name + "!")

# Function Calls
greet_with_default("Alice")  # Output: Hello, Alice!
greet_with_default()          # Output: Hello, Guest! (default value used)
